# threejs-fireworks

Three.js と Tween.js を使用して作成した花火アニメーションです。

打ち上げから爆発までをリアルタイムで描画し、色や打ち上げ位置をランダムに生成することで、毎回異なる花火を楽しめます。

🎆 デモ:
https://yoshida-png-oss.github.io/threejs-fireworks/

## 使用技術

- TypeScript
- Three.js
- Tween.js
- Vite

## 機能

- 花火を時間差で100発打ち上げ
- 爆発色をランダム生成（HSL）
- 打ち上げ位置をランダム生成
- パーティクルによる爆発エフェクト
- 導火線を表示し、打ち上げ中に徐々に短くなるアニメーション
- Additive Blending を利用した発光表現

## 工夫した点

### 導火線の表現

花火が上昇するにつれて導火線が徐々に短くなるように実装しました。

打ち上げの進行率から導火線の長さを計算し、毎フレーム頂点座標を更新しています。

### 爆発エフェクト

花火は約300個のパーティクルで構成されています。

球状にランダムな方向へ飛散させることで、本物の花火のような広がりを再現しました。

### 発光表現

Canvasで円形グラデーションのテクスチャを生成し、PointsMaterialへ適用しています。

さらに Additive Blending を利用することで、光が重なった部分がより明るく見えるようになっています。

### 色の変化

パーティクルが中心から離れるほど明るさを減衰させ、爆発後に自然と暗くなるようにしています。
